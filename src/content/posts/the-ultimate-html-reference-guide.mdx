## I. Document Foundation
### 1. Core Structure
```html
<!DOCTYPE html> 
<html lang="en"> 
<head> 
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Title</title>
</head>
<body> 
  <!-- Content goes here -->
</body>
</html>
```
