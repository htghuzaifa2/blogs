Updated 17 August 2025 – verified links, SMS codes, and complaint channels.

### 1️⃣ Eligibility Check in 30 Seconds

| Channel             | Step-by-Step                                       | Cost          |
| ------------------- | -------------------------------------------------- | ------------- |
| Official Web Portal | 1. Go to https://8171.bisp.gov.pk <br/> 2. Enter CNIC → click “Check Status” | Free          |
| SMS (Offline)       | 1. Open Messages <br/> 2. Type CNIC (no dashes) → send to 8171 | ₨ 2.5 + tax   |
| Alt Portal (Traffic Jam) | Use https://hunarmand.bisp.gov.pk → same CNIC flow  | Free          |

### 2️⃣ 2025 Eligibility Criteria (Quick Scan)

| You qualify if…                 | You do NOT qualify if…                         |
| ------------------------------- | ---------------------------------------------- |
| • Monthly income ≤ ₨ 60,000      | • You or spouse is a govt. employee            |
| • No family member abroad       | • Own > 3 acres irrigated land                 |
| • No vehicle registered in your name | • Have a passport / sponsored foreign travel |

### 3️⃣ New Registration (If Ineligible)

| Method               | What to Do                                                    |
| -------------------- | ------------------------------------------------------------- |
| SMS Registration     | Send CNIC to 8171 → receive “Under Review” reply              |
| BISP Tehsil Office   | • Original CNIC <br/> • Children’s B-Forms <br/> • Utility Bills |
| Online NSER Survey   | Fill at https://nsurvey.bisp.gov.pk (mobile-friendly)          |

### 4️⃣ Submit Complaints (Multiple Ways)

| Type                  | How                                       | Timeline            |
| --------------------- | ----------------------------------------- | ------------------- |
| Payment not received  | • WhatsApp by province (see table) <br/> • Call 0800-26477 | 2 working days      |
| Wrong eligibility     | • Web Portal → “Grievance” <br/> • BISP Tehsil Office | 7 days              |
| Fraudulent deduction  | • PTA 9000 (forward SMS) <br/> • FIA 1991 (cyber-crime) | Immediate block & refund |

### 5️⃣ Track Your Complaint

| Tool       | How                                          |
| ---------- | -------------------------------------------- |
| PTA CMS    | complaint.pta.gov.pk → enter tracking ID     |
| BISP Portal| Login → “My Complaints”                       |
| SMS        | Send “TRACK <ticket-id> to 8171              |

### ✅ One-Screen Checklist

1. Check eligibility: 8171.bisp.gov.pk OR SMS 8171
2. If eligible → collect payment via **HBL Konnect / JazzCash**
3. If issues → WhatsApp province number OR 0800-26477
4. Track via ticket ID
