“Your FIR is now just one CNIC away—no police station visits required.”

Below are step-by-step links and apps to view, download, or verify First Information Reports for Punjab, Sindh, Islamabad (ICT), and beyond.

### 🟢 **Punjab Police – PRS & ICTP Apps**

| Tool             | Steps                                                              | Direct Link / Download           |
| ---------------- | ------------------------------------------------------------------ | -------------------------------- |
| **PRS Portal**   | 1. Go to prs.punjabpolice.gov.pk<br/>2. Select FIR → District → Police Station<br/>3. Enter CNIC/FIR No. | Web only                         |
| **ICTP Mobile App** | 1. Install “ICTP” (Android / iOS)<br/>2. Register with CNIC<br/>3. Tap “View FIR” | [Play Store](https://play.google.com/store/apps/details?id=pk.gov.pitb.police.ictp) / [App Store](https://apps.apple.com/pk/app/punjab-police-pakistan/id1292837346) |

Punjab covers Lahore, Rawalpindi, Multan, Faisalabad, Gujranwala & all districts.

### 🔵 **Sindh Police – PRVS Portal**

| Tool          | Steps                                                                         | Direct Link                      |
| ------------- | ----------------------------------------------------------------------------- | -------------------------------- |
| **PRVS Portal** | 1. Visit prvs.sindhpolice.gov.pk<br/>2. Enter FIR No., District, and Year<br/>3. Click “Search” | [prvs.sindhpolice.gov.pk](https://prvs.sindhpolice.gov.pk/) |

Covers Karachi, Hyderabad, Sukkur, Larkana & other Sindh districts.

### 🟠 **Islamabad (ICT) Police**

| Tool                     | Steps                                                                      | Direct Link / App                                                                |
| ------------------------ | -------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| **ICT Complaints Portal** | 1. Go to complaints.islamabadpolice.gov.pk<br/>2. Login → “My FIRs”        | [complaints.islamabadpolice.gov.pk](https://complaints.islamabadpolice.gov.pk/)      |
| **ICT Police App**       | Same flow as Punjab’s ICTP (shared code base).                             | [Play Store](https://play.google.com/store/apps/details?id=com.pitb.ictp)        |

### 📄 Universal One-Screen Checklist

```
1. Keep your **CNIC** or **FIR number** ready.
2. Choose your province:
   • Punjab / ICT → **ICTP app** or **PRS web**
   • Sindh → **PRVS portal**
3. Enter details → **Download PDF** → share/print.
```

Bookmark this page and check tonight—no police station visit required.
