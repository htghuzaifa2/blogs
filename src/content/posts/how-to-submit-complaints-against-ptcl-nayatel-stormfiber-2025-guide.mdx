**Check outage alerts:**

### üìù ISP-Specific Complaint Channels

#### 1. PTCL (Pakistan Telecommunication Company Limited)

-   **WhatsApp:** Message `+92 331 218 1218` ‚Üí Send "Complaint" ‚Üí Follow prompts.
-   **Self-Service Portal:** Visit `ptcl.com.pk/Support` ‚Üí "Register Complaint". Enter Service ID ‚Üí Describe issue ‚Üí Attach proof.
-   **Email:** `complaint@ptcl.net` with subject: `[Service ID] - [Issue] - [Urgency]`.
-   **Helpline:** Call `1218` ‚Üí Demand escalation code.
-   **Social Media:** Tag `@PTCLOfficial` on Twitter/Facebook + complaint ID ‚Üí 3x faster response.

#### 2. Nayatel

-   **App/Portal:** Use **Nayatel Care App** ‚Üí "Report Fault" ‚Üí Upload evidence.
-   **Email:** `customercare@nayatel.com` (CC `ceo@nayatel.com` for delays).
-   **Helpline:** `111-111-111` ‚Üí Request Tier-2 support.
-   **Walk-in:** Visit franchise (e.g., Islamabad: Blue Area) ‚Üí Get written acknowledgment.

#### 3. StormFiber

-   **Portal:** Submit via **StormFiber GRM** ‚Üí Select "Internet Fault".
-   **WhatsApp:** Message `+92 321 111 1111` with service ID + issue.
-   **SMS:** Type `SF<space>Issue` ‚Üí Send to `8008`.
-   **Twitter:** Tag `@StormFiberPK` ‚Üí Include speed test screenshots.
