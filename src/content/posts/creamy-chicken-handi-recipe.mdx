*Serves 4 | Prep: 15 mins | Cook: 25 mins*

>"A handi is a flavor cauldron – where cream, tomatoes, and spices emulsify into velvet under high heat."

### 🧂 Ingredients

| Item              | Quantity        | Key Notes                                 |
| ----------------- | --------------- | ----------------------------------------- |
| Chicken (boneless)| 750g (cubes)    | Thighs for juiciness                      |
| Tomatoes          | 4 (puréed)      | Ripe + red                                |
| Onion paste       | 1 cup           | Golden-fried then blended                 |
| Ginger-garlic paste| 1.5 tbsp        | Freshly ground                            |
| Heavy cream       | 1 cup           | 35% fat min.                              |
| Yogurt            | ¼ cup           | Whisked smooth                            |
| Butter + ghee     | 4 tbsp each     | Butter for creaminess, ghee for aroma     |
| Green chilies     | 4 (slit)        | Adjust heat                               |
| Kasuri methi      | 1 tbsp          | Crushed (non-negotiable!)                 |
| **Spices**        |                 |                                           |
| › Cumin seeds     | 1 tsp           |                                           |
| › Turmeric        | ½ tsp           |                                           |
| › Red chili powder| 1 tsp           | Kashmiri for color                        |
| › Coriander powder| 1.5 tsp         |                                           |
| › Garam masala    | 1 tsp           |                                           |
| **Garnish**       |                 |                                           |
| › Ginger juliennes| 2 tbsp          |                                           |
| › Fresh coriander | ¼ cup           |                                           |
| › Cream drizzle   | 2 tbsp          |                                           |

### 👨‍🍳 Step-by-Step Instructions

#### 1. Onion Paste (The Secret)
1. Fry 2 sliced onions in 2 tbsp ghee until dark golden (12-15 mins).
2. Blend with 2 tbsp water → smooth paste.

#### 2. Sear Chicken
1. Heat 2 tbsp ghee → sear chicken on HIGH 3 mins → remove.
2. ⚠️ Don’t cook through – just seal juices.

#### 3. Build Gravy
1. Same pot → melt butter + 1 tbsp ghee.
2. Add cumin → sizzle → ginger-garlic paste → sauté 30 sec.
3. Add onion paste → fry 3 mins until glossy.
4. Add tomato purée + all spices → cook 8 mins until oil separates and the masala is a deep crimson.

#### 4. Cream Fusion
1. Reduce heat → whisk in yogurt → cook 2 mins.
2. Add chicken + green chilies → simmer 5 mins.
3. Stir in cream → **DO NOT BOIL** (cream splits!).

#### 5. Finish Like a Chef
1. Turn off heat → add:
   - Kasuri methi (crushed between palms).
   - Garam masala.
   - 1 tbsp butter → stir gently.
2. Rest 5 mins (⚠️ essential for flavors to marry).

### 🌟 Restaurant Secrets

- **Onion Paste**: Dark frying creates a caramel sweetness that balances the tomatoes.
- **Cream Control**: Add cream OFF the heat; the residual warmth will thicken it without curdling.
- **Butter Finish**: Cold butter stirred in last creates a glossy "lacquer" effect.
- **Handi Advantage**: The porous walls of a clay pot concentrate flavors. A heavy-bottomed pan is a good substitute.

### 🔥 Pro Variations

| Style          | Twist                                                |
| -------------- | ---------------------------------------------------- |
| Afghani        | Add ¼ cup ground cashew paste                        |
| Butter Chicken | Replace cream with ½ cup tomato purée + ½ cup cream  |
| Spicy          | Add 1 tbsp chaat masala                              |
| Murgh Makhani  | Stir in 2 tbsp honey + 1 tsp kasuri methi            |

**Storage Tip**: Tastes better the next day! Reheat on low with a splash of milk.

### ✅ Common Mistakes & Fixes

| Issue         | Solution                                 |
| ------------- | ---------------------------------------- |
| Cream curdled | Temper cream with 2 tbsp gravy first     |
| Gravy too thin| Simmer uncovered for 5 mins              |
| Lacks depth   | Add 1 tsp kitchen king masala            |
| Too spicy     | Stir in 1 tsp sugar + extra cream        |
