“Medical errors are no longer swept under the rug—hospitals now answer to regulators, not just reputations.”

Below are all official, up-to-date channels (national + provincial) for lodging complaints against hospitals, doctors, nurses, or diagnostic centres—with direct links, fees, timelines, and templates.

### 1. Pakistan Medical Commission (PMC) – National Portal for Doctors & Admissions

| Use Case                                 | Channel                             | Fee & Timeline                                      |
| :--------------------------------------- | :---------------------------------- | :-------------------------------------------------- |
| Doctor negligence / fake degree          | `admission.complaints@pmc.gov.pk`   | Free, 30-day review                                 |
| Medical college admission fraud          | Same email + evidence               | Strict action incl. admission cancellation          |
| Portal                                   | `pmc.gov.pk` → Complaints           | Upload PDF evidence (≤2 MB)                         |

### 2. Federal Health Directorates & Territories

| Territory      | Portal / Email               | Address / Phone                       |
| :------------- | :--------------------------- | :------------------------------------ |
| Islamabad (IHRA) | `ihra.gov.pk/complaints`     | DMLC Building, H-8/2, Islamabad       |
| AJK            | `ajkhealth.gov.pk`           | Muzaffarabad                          |

### 3. Provincial Healthcare Commissions

#### Punjab Healthcare Commission (PHC)

| Action            | Link / Details                                          |
| :---------------- | :------------------------------------------------------ |
| Online Complaint  | `os.phc.org.pk/complaintsform_patient.aspx`             |
| Office            | 185 New Garden Town, Lahore, 042-99333161-68             |
| Required Docs     | CNIC, medical records, receipts, duly signed complaint form |

#### Sindh Healthcare Commission (SHCC)

| Action        | Link / Details                                                    |
| :------------ | :---------------------------------------------------------------- |
| Form Download | `shcc.org.pk` → Download Form                                     |
| Address       | Suite 309, Block 09, Clifton, Karachi                             |
| Process       | 30-day written notice to hospital first → then notarized affidavit + form |
| Fine Power    | Up to ₨ 500,000 on hospital; ₨ 200,000 on false complainant         |

#### Khyber Pakhtunkhwa Health Care Commission (HCC-KP)

| Action           | Link / Details                          |
| :--------------- | :-------------------------------------- |
| Form + Affidavit | `hcc.kp.gov.pk` → Download Regulations  |
| Deadline         | 60 days from incident                   |
| Address          | HCC-KP Office, Peshawar                 |

### 5. Mobile Apps & Citizen Portal

| App / Portal            | What You Do                                                    |

| Pakistan Citizen Portal | Category: Health → Sub-category: Hospital / Doctor → upload evidence, track in real-time |
| PTA CMS                 | Use “Medical Services” for telemedicine fraud                  |

### 6. Complaint Template (Copy-Paste)

```
Subject: Complaint Against Dr. [Name] / Hospital [Name]

1. Complainant: [Your Name, CNIC, Phone]
2. Hospital/Doctor: [Full Name & Address]
3. Incident Date: [DD-MM-YYYY]
4. Issue: [Negligence / Malpractice / Fake Degree / Over-billing]
5. Evidence: [Medical records, receipts, photos, lab reports]
6. Relief Sought: [Refund, license suspension, compensation]
```

### 7. Timeline & Penalties (2025)

| Body             | Max Days | Fine / Action       |
| :--------------- | :------- | :------------------ |
| PMC              | 30 days  | License suspension  |
| Provincial HCC   | 60 days  | ₨ 50,000–500,000    |
| Citizen Portal   | 21 days  | Auto-escalation     |

### 8. One-Screen Cheat-Sheet

```
1. Identify jurisdiction (PMC / HCC / District)
2. Download form → fill + attach evidence
3. Submit online OR courier with notarized affidavit
4. Track via portal / SMS
5. Escalate after deadline
```
